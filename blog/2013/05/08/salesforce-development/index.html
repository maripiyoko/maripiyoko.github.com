Salesforce Developer試験

May 8th, 2013

Developer試験種類

以下の3種類

  • 認定デベロッパー試験
  • 認定上級デベロッパー試験
  • 認定テクニカルアーキテクト試験

申込方法（共通）

  • ネットから試験会場・日時を選択し、クレジットカード払いで予約
  • 試験日当日は会場にて、PCを使った選択問題（上位資格は二次、三次試験有り）
  • 関東での開催
    • セールスフォース・ドットコム直営会場
      • 六本木会場 （開催日: 不明）
      • 品川会場 （開催日 : 4月17日、5月14日、5月20日、6月4日、6月21日、7月3日、7月19日）
    • その他一般のテスト会場もある。ネットで登録時に確認可

認定デベロッパー試験

  • 設問数／時間 : 60問／90分
  • 受験料 : 21,000円
  • 試験方式 : 多肢選択式PC試験 → テストセンターで受験
  • サンプル問題、試験範囲

認定上級デベロッパー試験

  • 受験資格 : 認定デベロッパー試験合格者
  • 受験料 : 42,000円
  • 試験構成 : テストセンターで受験
    • 1次試験（多肢選択式PC試験 69問／120分）
    • 2次試験（プログラミング実技試験＋論述式試験）
  • 注意点 : 論述は現在(2013/5時点)英語のみで日本語対応については検討中

認定テクニカルアーキテクト試験

  • 受験資格 : 認定デベロッパー試験合格者
  • 受験料 : 無料 ＋ 52,500 円 ＋ 630,000 円
  • 試験構成 :
    • １次試験（オンラインで自己評価テスト 42 問／30 分（無料））
    • 2次試験（多肢選択式PC試験 60 問／120 分）
    • 3次試験（レビューボード面接試験 約 4.5 時間）
  • 注意点 :
    • 3次試験は2次試験合格後に改めて申込
    • レビューボードの約３週間前に送られてくる案内に従って顧客プレゼン資料を作成
    • 過去に携わった導入プロジェクトについて発表するため、顧客の承認が必要（承認フォーム提供）

リリース試験

  • 資格保持のため、Salesforceのアップデートに伴う更新試験を受け続けなければならない
  • 受験資格 : 認定デベロッパー試験合格者
  • 受験料 : 10,500 円（3リリース分込み）
  • 試験方式 : オンライン試験 約10問～15問／30分
  • オンライン試験なのでどこでも受験可能
  • 1年に3回リリースがあり、全ての試験を2個後のリリース試験が出るまでに受験・合格しないと資格失効する
  • 各リリース試験は3回まで追試可能

まとめ

  • ベース資格の、認定デベロッパー試験はすぐに取得できそう
  • 上位資格は、2次試験の英語論述や、面接試験等、実務経験豊富でないと合格は厳しそう
  • 資格保持に毎年10,500 円払う
  • 資格保持のためのリリース試験自体は簡単そう（しかも3回まで再試験可）

そもそもSalesforceとは？

  • 営業支援（SFA）、顧客管理（CRM）ソフト
  • カスタムアプリケーションの構築でより広い業務に適用できる
  • カスタムアプリケーションの導入事例 :
    • 日本郵政
    • ファーストリテイリング（ユニクロ）

オンライントレーニング

  • 無償提供のオンライントレーニングはSalesforceの使い方的な要素が多い
  • ビデオ内では画面が英語表記なので、Developer Editionで自分の言語を日本語に指定している場合、若干混乱する。

無償オンライントレーニング一覧と感想

はじめよう：Sales Cloud の利用

  • Salesforce初めての場合は見たほうが良い
  • Salesforceの一般的な使い方のイメージが掴める。（ある営業担当者の標準的な操作）
  • 大体どんなものかわかっている場合、不要。

はじめよう：Salesforce の操作

  • Salesforce初めての場合は見たほうが良い
  • 上記の続き。1つ目で大体イメージがつかめていれば不要。
  • 用語（英語）の説明があり良いかも。

はじめよう：Salesforce の管理基礎

  • 顧客の管理者向けの内容
  • 標準的なカスタマイズ可能箇所がわかり、見る価値あり。

はじめよう：レポート・ダッシュボードの利用

  • レポートの作り方など
  • さほど有用ではない

ビジネスにおけるソーシャルメディア：Chatter

  • 不要
  • Twitterに用語が似ている。普通に何かのソーシャルメディアを知っていれば見る必要なし。

はじめよう： Visualforce

  • 開発者向け内容
  • サンプルのアプリ作成方法がわかり、非常にオススメ
